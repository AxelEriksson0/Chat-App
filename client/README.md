## Docker

#### Build

>`docker build -t client:1.0.0 .`

#### Start
>`docker run -p 3000:80 --name chat_client client:1.0.0`

#### Access
Your client will be available on `http://localhost:3000/` or if running Windows you might have to find the IP address with `docker-machine ip default`.
